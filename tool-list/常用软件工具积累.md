# Win10使用技巧

| 快捷键                  | 说明                                                         |
| ----------------------- | ------------------------------------------------------------ |
| 【win+.】               | 按住win键+英文句号键，打开win10 自带的特殊符号输入框         |
| 【Ctrl+Win+方向箭头】   | Ctrl+win+方向右键或左键，快速切换虚拟桌面。只有多个虚拟桌面时，才可以使用该快捷键。 |
| 【win+←】               | 按住win键+方向键，可以让当前窗口随着方向键的切换进行上下左右停靠 |
| 【Win+Shift +方向箭头】 | 将窗口快速移至相邻的显示器                                   |
