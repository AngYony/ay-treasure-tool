# VSCode 插件推荐

### Code Runner

代码一键运行。

### Bracket Pair Colorizer 2

括号匹配颜色高亮显示。

### indent-rainbow

颜色区分代码缩进。

### vscode-pigments

css颜色标识。

### peacock

为每个vs code窗口配色。

### Better Comments

更好的代码注释功能

### output Colorizer

输出面板，提供语法高亮功能

### Debug Visualizer

可视化调试方式。

### Polacode

将选中的代码导出为图片格式，并保留代码的样式。
