# VS2026 关键配置



## 字体与颜色

颜色主题：选择系统设置

编辑器外观：浅色（额外对比度）

| 显示项名称                     | 设置值                        | 描述             |
| ------------------------------ | ----------------------------- | ---------------- |
| 大括号匹配、括号匹配（方括号） | 项背景：选择蓝绿色（#02ffff） |                  |
| 用户类型 - 类                  | 项前景：绿色，偏暗            |                  |
| 用户类型 - 委托                | 项前景：洋红色                |                  |
| 用户类型 - 接口                | 项前景：紫色，偏亮            |                  |
| 用户类型 - 枚举                | 项前景：橄榄绿，偏亮          |                  |
| 用户类型 - 类型参数            | 项前景：褐紫红色，偏亮        | 泛型中<T>T的颜色 |
| 用户类型 - 结构                | 项前景：绿色，偏亮            |                  |
